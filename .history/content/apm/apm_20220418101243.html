APM - 올인원 솔루션

Flexwill's
APM(Application Performance Management)
APM은 분산시스템에 대해 모니터링, 추적, 진단하는 APP 성능 모니터링 시스템입니다.


APM은 Storage로 Elasticsearch를 사용하여 Elasticsearch를 이용한 연관 분석, 집계 시각화를 통해 새로운 insight를 빠르게 얻습니다.
기존 오픈소스들의 약점을 해결하고 새로운 시작의 요구를 수용합니다.
Flexwill's APM
분산추적, 서비스 메시 원격분석, 메트릭스 집계 및 시각화를 위한 올인원 솔루션을 제공

마이크로 서비스, 클라우드 네이티브 및 컨테이너 기반( Docker, Kubernetes, Mesos) 아키텍처용으로 설계된 분산 시스템용 APM입니다.
Flexwill's
사용자 환경을 논리적으로 분해한 APM 올인원 솔루션
논리적으로 Probe, Backend, Storage, UI로 구성
  • Probe는 데이터를 수집하고 APM
    요구사항에 맞게 다시 포맷
  • Backend는 데이터 집계, 분석 및
    스트리밍 프로세스를 지원하며 Traces, Metrics, logs를 처리.
  • Storage는 개방형(플러그형)
    인터페이스를 통해 APM 데이터를 보관.
  • UI는 사용자가 APM 데이터를 시각화하고
    관리할 수 있는 웹 기반 인터페이스.
Storage로 Elasticsearch를 사용.
  • 데이터를 수집, 검색 및 분석할 수 있는 데이터 플랫폼으로 Elasticsearch를 이용한 연관 분석, 집계 시각화를 통해 새로운 insight를 빠르게 얻음.
데이터 문제 해결을 위한 플랫폼
ELASTIC STACK
  • Elasticsearch는 기존 오픈소스들의 약점을 해결하고
    새로운 시작의 요구를 수용.
다양한 애플리케이션 환경 지원
  • 데이터를 수집하고 APM 요구사항에 맞게 다시 포맷.
  • 애플리케이션 소스 변경 없이 Agent 구동으로 인한 overhea를 최소화하도록 튜닝.
  • 비동기 처리 및 gRPC 전송으로 Agent 구동으로 인한 overhead를 최소화하도록 튜닝.
플렉스윌의 최신 기술력으로
구축된 다양한 플랫폼을 통해
최상의 성능을 제공하여 고객의 가치 창출에 이바지 하고 있습니다.
Flexwill's
APM 올인원 솔루션의 다양한 기능
실시간 토폴로지와 대시보드를 한 번에 볼 수 있는 UI 제공
복잡한 시스템 전체를 쉬운맵으로 시각화하여 토폴로지노드의 상태 와 노드간의 호출 상태를 한 눈에 파악.
관심 서비스 및 성능 분석, 목적에 따라 커스터마이징
All, 서비스그룹별, 서비스별 대시보드를 확인할 수 있으며, 각 대시보드는 관심 서비스 및 성능 분석 목적에 따라 커스터마이징.
토폴로지의 서비스의 상세 정보를 Metrics 차트로 제공
서비스그룹별 토폴로지를 제공하며, 해당 서비스의 트레이스와 알람을 바로 확인.
프로파일링은 상호 작용 기능으로 Method 단계의 성능 진단을 제공
사용자가 생성한 프로파일 작업이 완료되면 프로파일링된 트레이스가 표시, 트레이스의 스택 기반 분석 결과를 통해 가장 느린 Class Method를 확인.
트레이스는 Endpoint의 수행단계를 다양한 형태로 제공
트레이스는 서비스, 인스턴스, Endppoint 및 호출에 대한 Metrics를 이해할 수 있도록 자세한 로그 형식, 연계된 분산 시스템의 트레이스도 함께 추적.
로그는 서버 로그를 수집하여 제공하며 트레이스와 연계
서비스에 적용한 logback log framework을 통해 서버 로그를 수집, 수집된 로그는 트레이스와 연계하여 분석 활동을 지원.
로그와 연계하여 해당 트레이스에 대한 로그 상세 내용 확인
Call Tree의 각 항목인 Span 정보를 통해 Component, package, Arguments, SQL문, Error stack 등을 확인.
알람에서는 트리거된 모든 알람 확인
서비스, 엔드포인트, 데이터베이스에 대해 평균 응답 시간, 성공률 등에 대한 알람 규칙을 생성. 알람 규칙에 의해 발생한 알람은 화면에서 조회.
대시보드는 여러 관점의 Metics를 다양한 형태로 시각화
서비스, 서비스 인스턴스, 데이터베이스, 엔드포인트의 처리량 (Calls), 응답시간, 성공률 등의 Metrics를 제공.
